# Word Counter
This Project is a very simple file reader that allows you to read, write, and view basic info about your file.


## Installation
None Required! simply run it in your VS code or in any code terminal. it is in Python and does import some extensions

## Usage
Use at as a bad Google doc. you can edit files quickly without having to open them. and guess what! it comes up with a timestamp and word counter when you need it.
